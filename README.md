# 越公子 - 面部美学设计师个人网站

> Sharp enough to be seen, soft enough to be felt.

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS 3.4
- GSAP + ScrollTrigger（滚动动画）
- Three.js（3D空间碎片特效）
- Lenis（平滑滚动）

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 设计亮点

- **移动端优先**的响应式设计
- 首屏 **"YUE"** 金色手写体描边动画
- 120个 Three.js 空间碎片飘动背景
- 液态玻璃质感 UI 元素
- GSAP 驱动的逐字出现和滚动触发动效
- 竖向箴言墙设计
